Accessibility Statement

This site has been designed to use Web Standards and aims to meet Web Content Accessibility Guidelines (WCAG) 2.0. (http://www.w3.org/TR/WCAG/) to A and AA standard.

In line with the requirements of the the Public Sector Bodies (Websites and Mobile Applications) (No. 2); Accessibility Regulations 2018 we are undertaking accessibility testing for WCAG 2.1 compliance. This site was developed in 2014 and if it is not compliant we will provide a roadmap towards compliance.

Web accessibility is an ongoing process and we are keen to improve accessibility on the site. If you are experiencing any problems using the site please do contact our helpdesk by emailing historicaltexts@jisc.ac.uk and we will do our best to assist.

Accessibility information for Historical Texts

Q. What is the maximum font size and does text reflow when you enlarge the font?

A. There is no maximum font size, and yes - text does reflow in the new responsive template.


Q. Can the background/foreground colours or contrasts be changed, and if so, how?

A. If the researcher uses a high contrast browser plugin like the one found here, then the site uses standard CSS so views will change to be high contrast.


Q. Are there keyboard-only equivalents for all mouse actions? Where could I find a list? Are there short cut keys to reduce tabbing round links?

A. No, only standard controls are available.


Q. Is text marked up so it can be navigated in a meaningful way (for example by heading level)?

A. Text markup is standard and semantic where possible.


Q. Can text be selected and read by text-to-speech tools?

A. Yes, with the proviso that Historical Texts is a Javascript application and the text-to-speech tools must understand DOM structure (so the text-to-speech tools must be recent).


Q. Are text descriptions available for all relevant graphics and images?

A. All graphics and images on our Home, About and Support pages now have text descriptions.


We hope this information is useful. For any enquiries, please contact the helpdesk at historicaltexts@jisc.ac.uk
