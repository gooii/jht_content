Quick Reference Guide

Last updated 9 October 2020

This is the online version of the Burney Collection Quick Reference Guide. If you wish, you can download this guide as a PDF file for offline viewing: Quick Reference Guide (PDF, 1953KB). You can click on any of the images below to see a bigger version for easier viewing.

This guide is for using the dedicated Burney Newspaper collection labs site (https://preview.historicaltexts.jisc.ac.uk).

The Quick Reference Guide aims to help users get started using the Burney Collection as quickly as possible. For more detailed help and guidance please visit our dedicated Burney Collection support pages.

If you have any queries or need more help, please email the Historical Texts helpdesk at historicaltexts@jisc.ac.uk.

The Burney Collection Home Page

Search the collection

Enter your search term in the search box. Choose which fields to search on using the all drop-down menu. The following options are displayed: All Fields, Title, Author, Printer/Publisher, Place of Publication, Bibliographic Number, Language, Illustration Description, Subject, Genre, Text type or Collection.

Note that the Burney Collection only contains metadata for the following fields: Title, Place of Publication, Bibliographic Number, Language and Text type.

Press enter or click the search button to conduct your search.

Search all of Historical Texts

You can go to the Historical Texts main site page by clicking the Search the whole of Historical Texts link.

Explore the collection

Explore the Burney Newspaper Collection by clicking the image links to a selection of articles.

Search View, with results shown below

Search results

The number of results created by your search is displayed on the search results page next to the search results box. You can access the advanced search options by clicking on the Advanced Search Options button.

The Histogram tool shows you a graphical distribution of your search results over time. You can click and drag to make a date selection and then use the scissor icon to finalise the selection.

Changing the search results display

Thumbnails are displayed next to each search result. These can be turned on or off using the Thumbnail images on and off buttons.

The results can be displayed as a list or as a grid. Change the view using the view mode buttons.

Search results are sorted by relevance by default but can also be ordered by date, author and title.

It is possible to change the number of results shown on the page using the results per page drop down menu. The side bar contains filters for Publisher, Author, Date, Subject, Text type, Genre and Collection. Clicking on these filters will cause the page to refresh as the filter is applied. Note that since there is only one collection in the labs site, although this filter can be applied it will not affect the results.

Biliographic records

Bibliographic data for the results is shown, including which collection it is from and whether the full plain text is available for it. If you are logged in, matches to your search term within the text are previewed.

Clicking the title or thumbnail of a result will open it in the Viewer.

Search View, expanded to show Advanced Options

Using the advanced search options

Expand your search results by adding fuzzy and variant spellings, misspellings and printer replacements.

You can add rows to include additional search terms and choose which fields they search on. The MUST, MUST NOT and SHOULD drop down option can be used with additional rows to create conditional searches.

Limit your search results by placing restrictions on date ranges, searching within the full text or description and whether a publication is illustrated. Note that since there is only one collection the labs site, using the collections selector may result in 0 results.

The reset button will restore the advanced search to its default state.

Use the Hide Search Options button to minimise the advanced search panel.

Use the My Saved Items tab to view and manage your saved items.

Using the Viewer to look at a publication

Navigation

The page image will be displayed in the central viewer pane.

Move through the page images of the publication and switch between single and double page views using the toolbar near the bottom-middle of the page.

Make the image larger or smaller by clicking on the magnifier icons or resize it using the resize button. Rotate the image in 90-degree increments using the rotate image button.

Viewer side panels

Click the Search button to open the left-hand search panel.

Click the Details button to open the right-hand details panel. The T icon indicates that the full text is available for this publication.

The Viewer, with the Search and Details panels open

Search panel

The pages tab contains matches to your search term in the document.

In the thumbnails tab, jump directly to a page image in the viewer by clicking on its thumbnail.

The publications tab shows other articles matching your search terms, allowing you to move between search results without leaving the viewer.

Details panel

The details tab shows the bibliographic record for the article.

The contents tab shows a table of contents if one is present.

The text tab shows the full OCR (machine-generated) text for the article.

The tags tab allows you to add personalised tags to an article for future reference. This feature can only be used if you are logged in using the UK Federation e.g. Shibboleth or OpenAthens

Other features

Click the share icon in the tool bar to copy a persistent URL link for the publication.

Use the download icon from the tool bar to download the full publication as a PDF file, a single page as a JPEG or the bibliographic details in the .RIS format.

Use the X to close and the handles to resize the side panels

The Search Panel, Viewing Panel and Details Panel can all be used simultaneously if desired.

Searching from within the Viewer

Using the pages tab

In the Pages tab of the search panel, you can enter a search term to find within the open document.

Fuzzy and variant spellings can be applied to the search and search results can be ordered by relevance or by page number. This may be particularly useful if the OCR for the full text is of poor quality.

Clicking on an instance of a word takes you to that page.

Click on the thumbnail or publications tabs to switch between these options

Navigating Through the Full Text in the Viewer

Using the details panel

View the full text of a publication and browse through the text using the back and forth arrows in the Text tab.

You can switch between plain and formatted text using the options below the navigation panel. If you have searched within the text, search terms will be highlighted in the plain text view.

Return to your search results using the results button in the header bar.
