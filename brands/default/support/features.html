Features by collection

Not all features are available for each collection. The search and display features we can provide depend on the data and bibliographic information provided to us, which varies by collection. The table below indicates where those differences lie so you know what's available for each collection.

Full text There is full text for around a third of EEBO titles. The text has been manually keyed by the Text Creation Partnership (TCP) and it is the same text that powers the ProQuest EEBO platform. Additional full text will be added to Historical Texts as it become available from the TCP. The full text available is formatted text. If full text is available for a record then the text "full text" will be displayed below the search result on the results page. There is full text for every title in the collection.
